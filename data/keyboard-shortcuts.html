O Policorp Linux possui muitos atalhos de teclado que ajudam a tornar o uso do seu computador mais eficiente.

A tecla Super é sua tecla Windows.

‌Atalhos do Policorp Linux

Alguns atalhos comuns usados no sistema operacional.

 F1 Abrir ajuda do GNOME.
 A Mostrar todos os aplicativos.

Janelas

Combinações de teclas que agilizam a manipulação e organização das janelas abertas no seu sistema operacional, permitindo alternar entre aplicativos, mover janelas e realizar outras tarefas de gerenciamento de forma eficiente.

 H Ocultar a janela
Alt  F10 Alternar estado de maximização
Alt  Espaço Ativar o menu da janela
Alt  F4 Fechar a janela
Alt  F7 Mover a janela
 Cima Maximizar a janela
 Baixo Restaurar a janela
 Esquerda Visualizar a divisão à esquerda
 Direita Visualizar a divisão à direita

Capturas de Tela

Combinações de teclas que permitem capturar instantaneamente imagens da tela do seu dispositivo, facilitando o compartilhamento de informações visuais importantes com apenas alguns cliques.

Shift  Print Fazer uma captura de tela
Alt  Print Fazer uma captura de tela de uma janela
Print Fazer uma captura de tela interativamente
Shift  Ctrl  Alt  R Fazer uma gravação de tela interativamente

Navegação

Refere-se ao uso de atalhos de teclado para percorrer rapidamente menus, pastas e aplicativos do sistema operacional, proporcionando uma experiência de uso mais fluida e produtiva.

 Tab Alternar entre aplicativos
 ' Alternar as janelas de um aplicativo
Alt  F10 Alternar estado de maximização
Alt  Esc Alternar as janelas diferentes
Alt  F6 Alternar as janelas de um aplicativo diretamente
Ctrl  Alt  Tab Alternar os controles do sistema
 Page Up Mover para o espaço de trabalho à esquerda
 Page Down Mover para o espaço de trabalho à direita
Shift   End Mover a janela para o último espaço de trabalho
Shift   Cima Mover a janela para o monitor acima
Shift   Baixo Mover a janela para o monitor abaixo
Shift   Direita Mover a janela para o monitor à direita
Shift   Esquerda Mover a janela para o monitor à esquerda

 L Bloquear tela
 N Focar na notificação ativa
 V Mostrar a lista de notificação
 S Mostrar o panorama
